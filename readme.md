# clone repository
```
git clone https://github.com/Cyri1/p7.git
```

# frontend

## Project setup : frontend folder
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

```
http://localhost:8080/
```


# backend

## Project setup : backend folder
```
npm install
```

### Compiles and hot-reloads for development
```
nodemon serve
```

```
http://localhost:3000/
```
